anchorsOnR moves to GitHub

The R-Packet anchorsOnR, which was created during my work as a working student at viadee, is now available on Github! With anchorsOnR one can create High-Precision Model-Agnostic Explanations for machine learning models. It allows for highly precise explanations in form of human readable IF-THEN rules, that describe which values caused the machine learning model's outcome for one specific observation.
